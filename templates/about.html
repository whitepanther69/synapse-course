About the Platform

A neurodivergent-first, multi-LLM AI tutor for secure coding.

From your first line of code to hands-on offensive and defensive security — guided by multiple AI models orchestrated through the Model Context Protocol (MCP), and designed accessibility-first for ADHD, dyslexia, and focused minds.

The engine

Powered by the Model Context Protocol (MCP)

At its core, SYNAPSE orchestrates multiple Large Language Models (LLMs) through a structured pipeline rather than relying on a single model. A primary tutor leads the conversation while additional models contribute specific steps — and you can also choose which model to work with directly in the workspace.

Technical note SYNAPSE coordinates several Large Language Models (LLMs) behind one tutoring workflow using the Model Context Protocol. Responses are produced through a defined multi-step pipeline — a primary model for the core explanation, with additional models contributing supporting steps — alongside manual model selection in the workspace and automatic fallback if a provider is unavailable. The goal is reliable, consistent guidance with no single point of failure.
How it teaches

A tutor that guides you, instead of just answering

SYNAPSE is built to protect your thinking — it helps you arrive at the answer rather than handing it over.

❓

It starts with questions

A Socratic opening: the tutor asks what you already know and what you think is happening, then adapts to your answers.

🪜

Hints in three stages

A scaffolded hint policy — a gentle nudge first, more specific only when you're stuck — withholding the full solution to prevent over-reliance.

🌿

Idle-aware encouragement

When you slow down, it offers a nudge, a different explanation, or a new approach. No pressure, no judgment — support when you need it.

🎨

More than one way to explain

If one explanation doesn't land, it switches styles — analogy, step-by-step, worked example — until it clicks.

Live practice

ShopSecure: a safe place to break things

ShopSecure is a deliberately vulnerable web application built for hands-on practice. It covers around 11 exploitable weaknesses spanning multiple CWEs across six OWASP Top 10 (2021) categories — path traversal, CSRF, insecure deserialization, injection, broken access control, and more.

You run real attacks, watch how the app responds, then fix the underlying code and see the impact immediately — the full detect → understand → remediate loop. It all happens in an isolated, auto-resetting sandbox with no external access, so every session starts clean and stays safe.

Inside SYNAPSE

What it actually feels like

You open a lesson, and a tutor starts a conversation — not a blank box waiting for input. You write code and it runs right away: Python in a secure server-side environment and Java in a protected sandbox, both with no setup, with risky operations blocked automatically so you can experiment safely.

And when you slow down, nothing rushes you. The tutor notices and offers a nudge, a fresh explanation, or a different angle — no pressure, no judgment. That's what accessibility-first learning feels like in practice.

Research

Grounded in research

SYNAPSE's design was informed by an early pilot involving neurodivergent and neurotypical learners, which returned encouraging usability results (SUS 76.4). The open-source replication package is archived under a Zenodo DOI.

The mission

Why SYNAPSE exists

Most learning platforms treat accessibility as a feature added at the end. Most AI tutors rely on a single model that does everything — well or poorly. SYNAPSE was built to challenge both assumptions.

The Problem

Technical education is often rigid, overwhelming, and built for one type of learner. Most AI tutors are monolithic — one model, one approach, one voice — and accessibility is an afterthought.

The Solution

SYNAPSE orchestrates multiple AI models through a Model Context Protocol (MCP) pipeline, pairing structured Socratic guidance with hands-on security practice.

Accessibility is built into the foundation — not added later — with dyslexia-friendly design, focus tools, dark mode, and learning support that adapts to each learner.

Who it's for

Made for everyone serious about learning

SYNAPSE is for anyone learning to code or strengthen their cybersecurity skills — students, junior developers, career switchers, hobbyists, and professionals.

It is especially designed for those who find traditional learning overwhelming, rigid, or difficult to follow — learners with ADHD, dyslexia, and other executive-function differences.

The platform provides a more structured and supportive experience, helping learners move forward with clarity and confidence.
